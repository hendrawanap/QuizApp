# QuizApp (Uplearn)
## Tugas Akhir Pemrograman Sistem Interaktif

### Deskripsi
Aplikasi ini adalah sebuah platform pembelajaran interaktif yang digunakan sebagai media belajar tentang makanan khas, ikon kota, dan wisata yang ada di Indonesia.

### Fitur Aplikasi
- Histori Kuis
- Kuis Pilihan Ganda & Isian Singkat
- Papan Peringkat
